# Web (Vizier's Vault marketing site)

Next.js app containing the public (home) marketing site: landing page, about, pricing, FAQ, team, contact, and legal placeholder pages.

## Scripts

- `bun run dev` – start dev server (Turbopack)
- `bun run build` – production build
- `bun run start` – run production server

## Environment (optional)

For SEO and metadata:

- `NEXT_PUBLIC_SITE_URL` – canonical base URL
- `NEXT_PUBLIC_SITE_NAME` – site name (default: Vizier's Vault)
- `NEXT_PUBLIC_SITE_DESCRIPTION` – default meta description
